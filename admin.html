📢 Admin Panel – New Body Fitness

This is a dedicated Admin Panel for managing the New Body Fitness website. It allows administrators to add, edit, and delete notices on the Notice Board. The panel provides a simple and intuitive interface where admins can:

  • Create new notices by entering a title and description.
  • Edit existing notices directly in the list.
  • Delete notices that are no longer relevant.
  • View all existing notices in one place.

The admin panel ensures that all updates are immediately visible to members, keeping the website content up-to-date and organized. Access to the panel is restricted to authorized users, maintaining security and control over the site content.
